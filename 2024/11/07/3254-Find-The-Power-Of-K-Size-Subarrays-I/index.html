3254.Find-The-Power-Of-K-Size-Subarrays-I

发表于2024-11-07|更新于2024-11-07
|阅读量:|评论数:

每日一题：

给你一个长度为 n 的整数数组 nums 和一个正整数 k 。

一个数组的 能量值 定义为：

  • 如果 所有 元素都是依次 连续 且 上升 的，那么能量值为 最大 的元素。
  • 否则为 -1 。
    你需要求出 nums 中所有长度为 k 的 子数组 的能量值。

请你返回一个长度为 n - k + 1 的整数数组 results ，其中 results[i] 是子数组 nums[i..(i + k - 1)] 的能量值。

示例 1：**
输入：nums = [1,2,3,4,3,2,5], k = 3

输出：[3,4,-1,-1,-1]

解释：

nums 中总共有 5 个长度为 3 的子数组：

  • [1, 2, 3] 中最大元素为 3 。
  • [2, 3, 4] 中最大元素为 4 。
  • [3, 4, 3] 中元素 不是 连续的。
  • [4, 3, 2] 中元素 不是 上升的。
  • [3, 2, 5] 中元素 不是 连续的。
    示例 2：**
    输入：nums = [2,2,2,2,2], k = 4

输出：[-1,-1]
示例 3：**
输入：nums = [3,2,3,2,3,2], k = 2

输出：[-1,3,-1,3,-1]

提示：

  • 1 <= n == nums.length <= 500
  • 1 <= nums[i] <= 10^5
  • 1 <= k <= n

标签
数组 滑动窗口

分析：

解法一：

直接模拟滑动窗口，窗口大小 < k的时候，移动右指针，窗口大小 = k时，移动双指针。如果发现不是连续的单调递增是时候，记录下当前的位置point。如果point出现在当前的区间里说明不满足单调性取-1，反之满足取最大值。

解法二：

参考高分题解，首先定义res长度，并初始化默认值为-1，这样可以方便后续直接修改最大值。记录当前最大的连续长度cnt，如果cnt > k，说明有cnt-k+1个坐标需要更新。所以当cnt > =k的时候就更新一下i+k-1位置上坐标，更新坐标时其实和cnt并没有什么关系，只和k有关。

例如：nums = [1,2,3,4,3]

当for循环遍历到第三个的时候，cnt=3，这时候可以更新i-k+1=2-3+1=0，

当for循环遍历到第四个的时候，cnt=4，这时候可以更新i-k+1=3-3+1=1。

所以cnt只是用来跟k做比较的，来执行是否进行替换操作。
